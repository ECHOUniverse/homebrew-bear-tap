# Homebrew Bear Tap

Homebrew Cask tap for [BearTodoMenuBar](https://github.com/ECHOUniverse/BearTodoMenuBar).

macOS menu bar utility that syncs Bear note checkboxes with system Reminders.

## Usage

```bash
# Add the tap
brew tap ECHOUniverse/bear-tap

# Install
brew install --cask bear-todo-menu-bar

# Upgrade
brew upgrade --cask bear-todo-menu-bar
```

## Requirements

- macOS 13 Ventura or later
- [Bear](https://bear.app/) app with Bear Notes
